17.1 Selection by label

For getting a cross section using a label:

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
%matplotlib inline
dates = pd.date_range('20190101',periods=10)
df = pd.DataFrame(np.random.randn(10,4), 
                  index=dates,

17.2 Selection by position

Select with the position of the passed integers:

df.iloc[2]
A    0.564102
B    1.448998
C    1.036779
D   -0.146018
Name: 2019-01-03 00:00:00, dtype: float64

By integer slices, acting similar to numpy/python:

df.iloc[2:4, 0:2]
A B
2019-01-03 0.564102 1.448998
2019-01-04 -0.939398 2.276844

By lists of integer position locations, similar to the numpy/python style:

df.iloc[[1,2,4],[0,2]]

For slicing rows explicitly:

df.iloc[1:3, :]

For slicing columns explicitly:

df.iloc[:,1:3]

For getting a value explicitly:

df.iloc[1,1]
